Q3 2024 focused on extensibility and search. July through September introduced the integrations marketplace, custom fields, and a powerful advanced search engine — making Camila significantly more adaptable to team workflows of all shapes and sizes.

### v1.8.0 — September 2024

- **Feature:** Integrations marketplace — connect Camila to Slack, Zapier, and Google Workspace directly from the Integrations settings page.
- **Feature:** Custom fields — add up to 20 custom fields per record type, with support for text, number, date, and dropdown field types.
- **Improvement:** File uploads now show real-time progress with an estimated time remaining, giving clearer feedback during large uploads.

### v1.7.1 — August 2024

- **Bug Fix:** Fixed broken OAuth redirect when users logged in via Google SSO from a private or incognito browser tab.
- **Bug Fix:** Fixed an issue where the "last seen" user timestamp was not updating on API-only interactions that didn't involve a browser session.
- **Improvement:** Password reset emails now arrive within 10 seconds, down from a previous maximum of up to 2 minutes.

### v1.7.0 — July 2024

- **Feature:** Advanced search — full-text search across all record types with boolean operators and field-specific filters for precise results.
- **Feature:** Data import — bulk import records from CSV with an interactive field-mapping step and a validation preview before committing.
- **Improvement:** Dashboard widgets are now drag-and-drop rearrangeable, allowing each user to organize their workspace to their preference.
