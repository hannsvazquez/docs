This page is an at-a-glance index of all new features added to Camila. Each entry links to its quarterly release notes page where you'll find full technical details.

<Note>
  Features are part of MINOR or MAJOR releases. Patch releases (x.x.PATCH) contain only bug fixes and small improvements.
</Note>

## 2025 Features

<CardGroup cols={2}>
  <Card title="Q2 2025" icon="sparkles" href="/changelog/2025/q2">
    **Advanced dashboard filtering** — Build multi-condition filters across any dashboard panel.

    **Bulk export** — Export thousands of records to CSV or JSON in a single action.

    **Custom notification rules** — Define exactly when and how Camila alerts you.

    **API rate limit headers** — Every API response now includes `X-RateLimit-*` headers for easier client-side throttling.
  </Card>
  <Card title="Q1 2025" icon="users" href="/changelog/2025/q1">
    **Team workspaces** — Separate, permission-controlled spaces for every team in your organization.

    **Webhook signature verification** — HMAC-SHA256 signatures on every outbound webhook payload.

    **Dark mode** — A fully accessible dark theme available across the entire Camila UI.

    **Keyboard shortcuts** — Navigate and act without leaving the keyboard. Press `?` to see the full map.
  </Card>
</CardGroup>

## 2024 Features

<CardGroup cols={2}>
  <Card title="Q4 2024" icon="code" href="/changelog/2024/q4">
    **Unified API v2** — A redesigned, consistent REST API with improved error messages and pagination.

    **Real-time collaboration** — See teammates' cursors and edits live on any record.

    **Saved views** — Save and share filtered, sorted views with your team.

    **Audit log** — A tamper-evident log of every action taken in your workspace.
  </Card>
  <Card title="Q3 2024" icon="puzzle-piece" href="/changelog/2024/q3">
    **Integrations marketplace** — Connect Camila to 50+ third-party tools in one click.

    **Custom fields** — Add text, number, date, select, and relation fields to any record type.

    **Advanced search** — Full-text search with field-level filters and boolean operators.

    **Data import** — Import records from CSV, Excel, or a connected integration.
  </Card>
  <Card title="Q2 2024" icon="shield" href="/changelog/2024/q2">
    **Two-factor authentication** — Secure every account with TOTP or SMS-based 2FA.

    **API key management** — Create, rotate, and revoke API keys from the settings dashboard.

    **Reporting dashboard** — Pre-built and custom charts across all your Camila data.

    **Comment threads** — Threaded discussions on any record, with @mentions and reactions.
  </Card>
  <Card title="Q1 2024" icon="repeat" href="/changelog/2024/q1">
    **Recurring tasks** — Schedule tasks to repeat on daily, weekly, monthly, or custom cadences.

    **Email-to-record** — Forward any email to a unique address to create a record automatically.

    **Public sharing links** — Share a read-only view of any record or dashboard with an external URL.

    **Multi-workspace support** — Belong to and switch between multiple Camila workspaces with one login.
  </Card>
</CardGroup>
