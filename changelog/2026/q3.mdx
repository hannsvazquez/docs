### July 19th, 2026

**Improvement** — **WhatsApp message delivery status indicators**

**Product Category:** AI Workflows

Sales agents can now see real-time delivery status on every outbound WhatsApp message directly from the Contact detail page. Each message displays one of four status labels: **Enviado** (Sent), **Entregado** (Delivered), **Leído** (Read), or **Fallido** (Failed). Follow-up templates triggered by Camila's automation also display a human-readable label — for example, _Plantilla de Seguimiento 04 · Ver contenido en WhatsApp Manager_ — instead of the raw template key stored in the system.

This improvement is backed by two new Supabase schema additions: new columns (`outbound_message_status`, `wamid`, `is_follow_up`) on the `whatsapp_messages` table and a new `whatsapp_status_events` table that tracks real-time delivery receipts using the WhatsApp Business API's message ID (`wamid`).

**How to try:** Open any Contact → WhatsApp tab. Each outbound message now shows a colored status badge next to the timestamp. Automated follow-up messages display their template name alongside a link to view content in WhatsApp Manager.
